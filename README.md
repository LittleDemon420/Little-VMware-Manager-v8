Little VMware Manager 🛠️⚡
Es una herramienta de interfaz de línea de comandos (CLI) ligera e interactiva diseñada para entornos Windows. Su objetivo principal es centralizar, automatizar y simplificar la gestión de los servicios, procesos y configuraciones de red de VMware Workstation, todo desde una única consola unificada y con un diseño de estilo retro/ASCII.

Evita tener que lidiar con la consola services.msc de Windows o recordar comandos complejos; este script proporciona un panel de control con accesos directos rápidos para optimizar tu flujo de trabajo de virtualización.




✨ Características Principales


1.Control Total de Servicios: Inicia, detiene o reinicia los servicios de VMware de manera global con un solo comando.


2.Kill Switch Integrado: Forzado de cierre inmediato de procesos colgados para liberar memoria RAM y desbloquear la aplicación.


3.Gestor de Redes Virtuales: Acceso directo al Virtual Network Editor, reinicio de adaptadores y diagnóstico rápido de IPs.


4.Automatización del Modo de Arranque: Configuración rápida para alternar entre el modo manual (ahorro de recursos cuando no usas VMware) y el modo automático (siempre listo).


5.Dashboard Informativo: Muestra en tiempo real metadatos clave del sistema como la versión del Kernel, usuario activo, IP asignada y el estado de los servicios.


6.Herramientas de Sistema: Accesos directos integrados para invocar PowerShell, CMD o el Administrador de Tareas con privilegios administrativos.



⚠️Importante⚠️

1.Para que todo funcione correctamente VMware tendra que ser instalado sin modificar rutas de fabrica.


2.Debe ejecutar el archivo con Privilegios de Administrador.
